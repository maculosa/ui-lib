## Props

| 参数     | 说明               | 类型     | 默认值 |
| -------- | ------------------ | -------- | ------ |
| text     | 文本               | `string` | -      |
| color    | 文本颜色           | `string` | -      |
| fontSize | 文字颜色           | `number` | -      |
| gap      | 水印图片之间的间距 | `number` | -      |
| fullscreen | 是否全屏           | `boolean` | `false` |
