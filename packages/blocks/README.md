# @banmao/blocks

基于 Naive UI 封装的登录组件。

## 特性

- 🚀 开箱即用的高级组件
- 💪 基于 Naive UI，享受其完善的基础组件能力
- 📦 支持按需加载，减小打包体积
- 🎨 继承 Naive UI 的主题系统

## 安装

```bash
# 安装 @banmao/blocks
pnpm add @banmao/blocks
```

## 使用

```vue
<script setup lang="ts">
import { Login } from '@banmao/blocks'

</script>

<template>
<Login />
</template>
```

## 组件列表

- Login - 登录表单组件
